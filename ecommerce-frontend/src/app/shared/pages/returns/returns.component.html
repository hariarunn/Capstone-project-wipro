Returns & Refunds

Simple, hassle-free returns on eligible items.

Eligibility

  • Return within 7 days of delivery.
  • Item must be unused, with original packaging & tags.
  • Non-returnable: perishable goods, gift cards, and items marked “Final Sale”.

How to start a return

  1. Open your order in My Orders.
  2. Choose Start return and select a reason.
  3. Pick pickup or drop-off, then confirm.

Refunds

Once we receive and inspect your item, refunds are issued to the original payment method within 3–5 business days.
